# ROBO-CAR-USING-ARDUINO
Autonomous Robo Car built with Arduino, servo, and ultrasonic sensors. Programmed microcontroller logic for obstacle detection and real-time navigation. Showcases skills in embedded systems and hardware-software interfacing.

This project details the development of a fully autonomous mobile robot using an Arduino Uno, designed to navigate environments by detecting and avoiding obstacles in real-time.
